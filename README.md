SkyAdmin
===============

### 相关项目
* [SkyAdminApi](https://github.com/Sky9th/sky-admin-api) PHP后端
* [SkyAdminVue](https://github.com/Sky9th/sky-admin-vue) Vue2.0前端

## 主要功能

* RBAC模型权限控制
* 文件管理 (开发中)
* 图片上传、压缩、裁剪（开发中)
* 微信、小程序管理及接入 (开发中)
* Swoole对接 (计划)

## 安装

~~~
npm install
~~~
